# Infiltrated: Escape from the Cult
### História

Você é um infiltrado, um agente enviado para investigar a Mansão, sede de uma seita ocultista conhecida como Os Filhos do Limiar. O que deveria ser uma missão de reconhecimento vira uma armadilha: os rituais da seita corromperam a própria arquitetura do lugar.
A mansão não obedece mais à geometria normal. Cada vez que você atravessa uma passagem, os cômodos se reorganizam , como se o edifício estivesse vivo e te testando. No centro de tudo está O Limiar, um artefato de poder inexplicável que os cultistas usam para dobrar o espaço da mansão à vontade. Você precisa chegar até ele antes que o ritual seja concluído, ou a mansão se fechará para sempre com você dentro.

Salas
A mansão é composta por 5 salas por run. A primeira é sempre a mesma, o Salão de Entrada, onde você ainda tem controle do ambiente. A partir daí, O Limiar embaralha tudo: as próximas 3 salas são escolhidas aleatoriamente entre ambientes como a Sala de Rituais, a Biblioteca Proibida, o Corredor das Velas e a Câmara de Reunião. A quinta e última sala é sempre o confronto com o Boss, o guardião designado pela seita para proteger o artefato.

Jogabilidade
O jogo é um top-down shooter roguelike. Você se move pelos ambientes eliminando cultistas e sobrevivendo até o portal de saída abrir.
Combate: o player dispara projéteis matando inimigos dispostos aleatoriamente nas salas. Cada abate concede 100 pontos.
Inimigos por sala: as salas escalam em dificuldade, começando com 6 inimigos e podendo chegar a 14 nos andares mais avançados.
Upgrades entre salas: ao fim de cada sala (exceto a do boss), você escolhe um entre três upgrades:

Portal de saída: após eliminar todos os inimigos de uma sala, um portal aparece para levar você à próxima.
